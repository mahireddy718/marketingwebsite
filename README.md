# ✨ Features

### 🧠 AI-Powered Insights

- **Smart Search:** Context-aware product discovery using Google Gemini AI  
- **AI Skin Analyzer:** Personalized skincare recommendations  
- **AI Routine Builder:** AI generated daily wellness guides  
- **Ingredient Checker:** Analyze product composition instantly

---

### 🛡️ Secure & Seamless Experience

- **Hybrid Authentication:** Google OAuth + JWT  
- **Reliable OTP Delivery:** Resend API integration  
- **Premium Checkout:** Razorpay payment gateway  
- **Real-time Support:** Socket.io chat system

---

### 💎 Boutique Aesthetic

- **Luxury UI Design:** Glassmorphism and premium typography  
- **Smooth Animations:** Framer Motion transitions  
- **Responsive SPA:** Optimized routing for Vercel deployment

# 🛠 Installation & Setup

## Prerequisites

- Node.js (v18+)
- MongoDB Atlas
- Google Gemini API Key
- Razorpay API Key

---

## Backend and Frontend Setup 

```bash
cd server
npm install

create .env file
PORT=4002
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret
RESEND_API_KEY=your_key
GEMINI_API_KEY=your_key
RAZORPAY_KEY_ID=your_id
RAZORPAY_KEY_SECRET=your_secret

RUN Backend
npm run dev

Frontend setup
cd frontend
npm install
npm run dev

project structure
frontend/
 ├── src/
 │   ├── components/
 │   ├── context/
 │   ├── pages/

server/
 ├── models/
 ├── routes/
 ├── utils/
